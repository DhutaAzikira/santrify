# Santrify

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Backend

1. composer install
2. php artisan key:generate  
3. php artisan cache:clear
4. php artisan migrate

php artisan serve
